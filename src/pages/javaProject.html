Invernadero Web


Aplicación web construida con Spring Boot que forma parte de un sistema de gestión para un invernadero

Permite administrar información relacionada con las plantas, permitiendo registrar plantas de tipos Orquideas, Cactus, Suculentas, Rosas y visualizarlas en el inventario, con filtros dimanicos que ayudan a manejar el inventario.
